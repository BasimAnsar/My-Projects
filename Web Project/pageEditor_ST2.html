Home Page

Technical Description

1. HTML Structure:

  • Navigation Bar (<nav class="nav-bar">): - A fixed navigation bar provides a user-friendly menu. - Includes a <img> for the logo and a title inside <div class="title"> for branding. - Navigation links are structured using <div class="tab">, each containing an <a> tag for seamless navigation.
  • Hero Section (<header>): - Features a background video using the <video> tag for an immersive experience. - A semi-transparent overlay (<div class="header-overlay">) ensures readability. - Headings (<h1> and <h2>) create a strong call to action. - A call-to-action button (<a class="cta-button">Join Us Now</a>) directs users to the volunteer page.
  • Content Sections:
    • Volunteer Section: - Introduces the mission to combat climate change. - Includes a descriptive paragraph, an image (<img>), and a button (<button>) for user interaction. - Uses the onclick event handler to navigate to the volunteer page.
    • Content Cards (.card-container): - Contains interactive cards, each linking to climate-related topics like Renewable Energy and Deforestation. - Cards use the <a class="card"> tag for navigation. - A hover effect (:hover) adds a slight elevation for a dynamic experience.
    • Quote of the Day Section: - Displays an inspirational quote inside a <blockquote> for semantic meaning. - Uses the quote-marks class to style large punctuation marks for emphasis.
    • Video Section: - Features a climate awareness video using the <video> tag. - Includes controls for play, pause, and volume adjustment.
  • Footer Section: - Includes navigation links, contact details (email & phone), and social media icons. - Features a newsletter subscription form for user engagement. - Uses icons (<i class="fab fa-facebook-f">, <i class="fab fa-twitter">) for social links. - Displays a copyright statement at the bottom.

2. CSS Styling:

  • Flexbox Layout: The website utilizes Flexbox for efficient alignment and spacing of elements like .grid-container and .navigation-bar. This ensures a dynamic and structured layout that adapts smoothly to different screen sizes.
  • Hover Effects: Media queries enable an adaptive design where .content-section restructures itself based on device width. Images and videos are optimized with max-width: 100% to maintain proportional scaling across all screens.
  • Colors and Readability: A modern, accessible color scheme is used, ensuring contrast and clarity. The primary color set (#2176ae, #1a5276) is chosen to complement an educational theme. High contrast between text and backgrounds improves readability for all users, including those with visual impairments.

3. Accessibility Considerations:

  • Alt Text for Images: Descriptive alt attributes are included for all images, such as alt="Climate Changers Logo", ensuring screen readers can convey meaningful context to visually impaired users.
  • Video Accessibility: The video element includes a fallback message ("Your browser does not support the video tag") for users with unsupported browsers. Adding captions or subtitles is recommended for better accessibility for hearing-impaired users.
  • Headings and Structure: Semantic HTML tags like h1, h2, h3, and footer improve content organization and readability for screen readers. Clear section headings make navigation easier for all users.
  • Buttons and Links: Interactive elements, such as buttons and links, are visually distinct and include hover effects. Descriptive link text (e.g., "Learn More" and "Join Us Now") improves usability for screen readers.
  • Keyboard Navigation: The website is fully navigable using a keyboard. Users can tab through links, buttons, and form elements. The tabindex="0" attribute ensures that interactive elements remain accessible for keyboard users.
  • Form Accessibility: Forms (such as the newsletter signup) include properly associated labels with input fields, ensuring compatibility with screen readers and improving usability for all users.

Statistic Page

Technical Description

1. HTML Structure:

  • Navigation Bar (nav-bar): The navigation bar (nav) contains links (<a>) to different pages of the website such as Home, Volunteer, Profile, etc. It allows users to navigate easily to various parts of the site. The navigation is visually structured with <div> elements to separate the logo and links for better organization. The alt attribute for the logo image (<img>) is set to "Climate Changers Logo" to enhance accessibility for screen readers, which is a good practice for users with visual impairments.
  • Header Section: The <header> includes a background video (<video>) that plays automatically and loops. The video is used to grab attention, highlighting the theme of the website â€” combating climate change. The header includes a main title (<h1>) "Climate Change Statistics" and a subtitle (<h2>) with a message "Global Warming is a Global Warning". These elements provide a clear introductory context to the website. The header also contains a call-to-action (CTA) button (<a href="volunteer.html" class="cta-button">Join Us Now</a>), encouraging users to volunteer.
  • Table Section: This section displays the key climate data in a table format. The <table> element is used to organize the data into rows and columns. The table also contains <caption> to describe the data in the table and <thead>, <tbody>, and <tfoot> sections for better organization. The table data includes indicators like COâ‚‚ emissions, temperature rise, and extreme weather events.
  • Chart Section: This section visualizes global COâ‚‚ emissions over time using a line chart rendered with Chart.js. The chart includes zoom and pan functionality provided by the chartjs-plugin-zoom plugin. A reset zoom button is available for users to reset the chart view.
  • Video Section: The video section allows users to watch a video titled "Are We Already Doomed?". It uses the <video> tag with controls so users can play, pause, and adjust the volume. This section visually supplements the content with a video related to the impact of climate change.
  • Footer Section: The footer includes links to other pages of the site, such as Home, Volunteer, Statistics, and Privacy Policy. It also includes contact information, social media links, and a newsletter subscription form.

2. CSS Styling:

  • Table Layout: The table is designed with proper spacing between rows and columns for clarity. It has alternating row colors for better readability. The table is responsive, with horizontal scrolling enabled on smaller screens using overflow-x: auto.
  • Chart Styling: The chart is made responsive and visually appealing, with custom colors and tooltips. The chartâ€™s line and points are styled with green shades to match the environmental theme. The chart supports zooming and panning to explore data more interactively.
  • Video Styling: The video container is styled with rounded corners and a shadow effect to ensure it blends well with the rest of the content. It is also responsive and adjusts for different screen sizes.
  • Hover Effects: The cards in the content section have hover effects, making them elevate and create an interactive feel. The hover effect adds an extra layer of user engagement.
  • Colors and Readability: The green color palette (#228b22, #1b691b) is used throughout the website to create a visually cohesive theme related to the environment. High contrast is maintained for readability, and dark text on light backgrounds enhances accessibility.

3. Accessibility Considerations:

  • Alt Text for Images: All images, such as the logo and table images, include descriptive alt attributes like alt="Climate Changers Logo", which ensures that screen readers can provide context for users who are visually impaired.
  • Video Accessibility: The video includes a fallback message ("Your browser does not support the video tag"). It is recommended to add captions/subtitles for the video to ensure hearing-impaired users can also engage with the content.
  • Table Accessibility: The table includes clear headers and captions that help screen readers interpret the data. It is structured with <th> elements for column titles and <td> elements for data points.
  • Buttons and Links: Buttons and links are visually distinct, with clear hover effects that help users identify interactive elements. Descriptive text is used in buttons like "Learn More" and "Join Us Now" to improve accessibility.
  • Keyboard Navigation: The website is designed to be fully navigable using a keyboard. Users can tab through links, buttons, and form elements. The tabindex="0" attribute is used in elements to ensure they are included in the tab order.

Quality Education

Technical Description

1. HTML Structure:

  • Navigation Bar (nav): The navigation bar (<nav>) is fixed at the top-right of the page, containing links (<a>) that allow users to navigate to the Home page. The navigation bar is styled with a red background, rounded corners, and a box shadow for a modern look. The active link is highlighted with a color change on hover.
  • Header Section: The header includes a main title (<h1>) "Quality Education" that emphasizes the focus of the page. The background image used behind the text adds visual appeal. The <h1> is styled with a serif font for a more academic look.
  • Container Section: The main content is housed in a container (<div class="container">) which includes the body of the page. This section contains the introduction to education, its barriers, and the solutions to overcome them. It uses an elegant layout with shadow and border-radius effects for modern aesthetics.
  • Content Images: Images are included in the content using <img> tags. These images are styled to have smooth hover effects (zoom effect) when the user interacts with them, providing a more engaging experience.
  • Barriers to Quality Education: A section highlighting major challenges such as inadequate access to education, gender inequality, infrastructure issues, poverty, and political instability. This section is organized with headings (<h3>) to distinguish each barrier.
  • Footer Section: The footer section is left blank, but it can be used to include contact information, social media links, or additional navigation links.

2. CSS Styling:

  • Background and Typography: The background image of the page is used for visual appeal, while the text is kept readable with sufficient contrast and appropriate font sizes. The primary font used is 'Poppins', with serif fonts like 'Lora' used for headings to create an academic theme.
  • Container Layout: The container is styled with a white background, rounded corners, and a subtle box shadow to create a card-like effect. It has padding and margin for spacing, and a z-index is used to ensure it appears above other elements.
  • Hover Effects: Images have a subtle zoom effect (transform: scale(1.03)) when hovered over, giving a modern interactive feel. Links are styled with hover effects that change color to enhance user interaction.
  • Separator Styling: The separators (<div class="separator">) are styled with a linear gradient background to visually separate sections. These sections also use slight opacity for a smooth transition effect.

3. Accessibility Considerations:

  • Alt Text for Images: All images used in the page include alt attributes for accessibility, such as alt="Education Image", ensuring the content is accessible for screen readers.
  • Header and Section Headings: The page uses clear and descriptive headings like <h1>, <h2>, and <h3> to improve the logical structure of the content for better navigation.
  • Link Accessibility: Links are styled with enough contrast and clarity to ensure they are easily distinguishable for all users. Links also include descriptive text for better understanding, such as "Learn More" or "Click here to visit the Statistics page."
  • Keyboard Navigation: The website ensures all interactive elements such as links and buttons are navigable via keyboard. The tabindex attribute is used where necessary.
